# auto-report-vue

為本人所在的NTUT MMS實驗室，為解決每日實驗室高度重複性的任務所設計的系統

`Auto Report`是一個成員進度管理系統，預計會實作以下功能：
- 設定提醒成員更新進度
- 設定每日寄送進度報表排程

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
